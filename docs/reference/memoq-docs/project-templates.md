# Project Templates in memoQ

> Source: https://docs.memoq.com/current/en/Concepts/concepts-project-templates.html (memoQ 12.2)

## Core Functionality

Project templates enable users to quickly generate local or online projects with pre-configured settings. The system stores workflow preferences and resource assignments to eliminate repetitive manual setup.

## Key Features

### Pre-defined Configurations
- Resource settings (TMs, TBs, LiveDocs, etc.)
- User assignments
- Automated action sequences

### Automation Capabilities
When creating a template, users can set up pre-defined configurations like resources, users, and actions. Users can also automate workflow steps such as pre-translation and running statistics.

## Use Case

Templates facilitate client-specific project creation by preserving previously established configurations, reducing the need for manual post-project tasks like statistics generation, pre-translation, and resource allocation.

## Related Features
- Edit project templates
- Automated actions configuration
- Creating projects from templates (local and online variants)
