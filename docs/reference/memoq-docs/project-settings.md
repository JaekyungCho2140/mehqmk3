# memoQ Project Settings Reference

## Overview

The Settings pane in Project home allows users to view and modify project configurations.

## Access Instructions

1. From the Dashboard, create or open a project
2. In Project home, click Settings
3. The General tab opens by default

## Settings Tabs and Functions

### General Tab
View or change the languages, main details and other options of the project.

### Segmentation Rules Tab
Choose or create a segmentation rule set for the project, governing how text breaks into translatable segments.

### QA Settings Tab
Select or create a quality assurance profile for implementing project-specific quality checks.

### TM Settings Tab
Choose or create a TM settings profile to configure translation memory behavior and parameters.

### LiveDocs Settings Tab
Select or create a LiveDocs settings profile for corpus management configuration.

### Auto-translation Rules Tab
Choose or create auto-translation rule sets. You can choose auto-translation rules separately for each target language.

### Non-translatable Lists Tab
Choose or create non-translatable lists to identify content that should not be translated.

### Export Path Rules Tab
Select export path rules with options for individual files and folders. Rules for folders work for documents imported through the Import folder structure command.

### LQA Models Tab
Choose or change an LQA model for linguistic quality assurance.

### Font Substitution Tab
Select or modify a font substitution profile for display preferences.

### MT Settings Tab
Choose or change the MT settings for machine translation configuration.
