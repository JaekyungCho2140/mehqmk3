# Supported Languages in memoQ

> Source: https://docs.memoq.com/current/en/Concepts/concepts-supported-languages.html (memoQ 12.2)

## Language Code System

memoQ supports both 3-letter and 2-letter language codes. The main language code is the 3-letter one, used in file names by default.

## Source Language Restrictions

Ten languages function as **target-only** (translation INTO but not FROM), because these languages are written in a script that does not mark sentence boundaries:

- Amharic, Burmese, Karen, Kayah (Myanmar), Khmer, Lao, Mon, Rakhine, Shan, Tigrigna

## Special Requirements

**Thai** as source language requires TM+ (introduced in version 11.2).

## Language Coverage Highlights

- English variants for 11 regions
- Spanish variants for 20+ regions
- Arabic dialects for 13 countries
- Chinese, Portuguese, German, French regional variants
- Indigenous languages (Navajo, Lakota, Mohawk, Mi'kmaq, etc.)
- Constructed language (Klingon)
- Most languages supported since version 9.0
