# Filter Configurations in memoQ

> Source: https://docs.memoq.com/current/en/Concepts/concepts-filter-configurations.html (memoQ 12.2)

## Overview

Filter configurations allow for fine-tuning file filters, enabling customization of how various file formats are processed during import operations.

## Supported File Format Customization Examples

- **INX files**: Import index entries from Adobe InDesign files
- **Word documents**: Import table of contents
- **Excel spreadsheets**: Exclude specific cell ranges
- **XML files**: Import only segments marked with `translate="yes"`

## Access Method

Filter configurations are available through the **Documents ribbon** using **"Import With Options."** Users can access advanced settings by selecting **"Change filter and configuration"** during document import.

## Sharing Capability

Users can distribute their filter configurations with colleagues, which is particularly valuable for complex formats like XML and Excel that offer numerous configuration options.

## Key Points

- Filter configurations are a **light resource** type in memoQ
- They control how document import filters process files
- Each supported file format can have its own filter configuration
- Configurations can be saved, exported, imported, and shared
- All resource exchange formats in memoQ are XML-based
