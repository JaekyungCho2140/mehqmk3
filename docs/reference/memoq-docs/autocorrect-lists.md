# AutoCorrect Lists in memoQ

> Source: https://docs.memoq.com/current/en/Concepts/concepts-autocorrect-lists.html (memoQ 12.2)

## Overview

AutoCorrect lists enable automatic text replacement during translation work, streamlining typing for frequently used terms and expressions.

## Key Features

### Text Replacement
Substitutes abbreviations with full forms automatically. For instance:
- "FYI" becomes "For your information"
- "JB" expands to "John Brown"

### Capitalization Correction
- Missed capitalization at the start of words gets corrected automatically
- Accidental CAPS LOCK activation is fixed without user intervention

## Access Points

1. The **Resource console**
2. The **Translation settings** menu on the Translation ribbon > AutoCorrect option
