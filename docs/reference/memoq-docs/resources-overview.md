# Resources in memoQ — Complete Overview

> Source: https://docs.memoq.com/current/en/Concepts/concepts-resources.html (memoQ 12.2)

## Definition

Resources are pieces of information or data that are regularly used during translation or translation management, designed for reuse across projects and teams.

## Heavy Resources

| Resource | Description |
|----------|-------------|
| Translation memories | Segment pairs (source + target) |
| Term bases | Words/expressions glossaries |
| LiveDocs corpora | Reference documents, translated pairs |
| Muses | Predictive typing suggestions |

## Light Resources

| Resource | Description |
|----------|-------------|
| Auto-translation rules | Dates, numbers, legislation references |
| Segmentation rules | Text splitting rules |
| Non-translatable lists | Expressions to skip/copy as-is |
| Ignore lists | Spell checker exceptions |
| AutoCorrect lists | Abbreviation expansion |
| Filter configurations | Document import settings |
| Export path rules | File naming/location conventions |
| Keyboard shortcuts | Functionality binding |
| QA settings | Quality assurance checks |
| TM settings | Match qualification criteria |
| LiveDocs settings | Corpus match criteria |
| Stop word lists | Terminology extraction filters |
| Web search settings | Customizable search providers |
| LQA settings | Linguistic quality models |
| Font substitution settings | Export formatting |
| Project templates | Creation automation |
| MT settings | Machine translation customization |

## Format & Exchange

All resource exchange formats in memoQ are XML-based. Where appropriate, memoQ uses corresponding open standards.

## Management

Resources are managed through:
- **Resource console** — central management
- **Project home** — project-level settings

## Offline Access

Light resources work offline when using online storage; users must update online resources before going offline via the memoQ menu.
