# Supported Document Formats in memoQ

memoQ supports importing and exporting through "import filters" which are software components enabling file format support. To configure filters, navigate to Documents > Import > Import with Options, select a file, then access Document import settings to adjust configurations.

## Monolingual Formats

### Adobe Products
- **Adobe InDesign Exchange Documents** (.inx)
- **Adobe InDesign Markup Language** (.idml)
- **Adobe InCopy Documents** (.icml)
- **Adobe FrameMaker** (.mif, .mi2)
- **Adobe Photoshop** (.psd)

### Web & Markup Languages
- **HTML Documents** (.html, .htm)
  - Server Side Includes (.shtml)
  - ASP.NET (.aspx)
  - Jakarta Server Pages (.jsp)
  - PHP files (.php)
- **XML Files** (.xml, .sgm, .sgml, .ttml)
- **Markdown** (.md)
- **YAML** (.yaml, .yml)
- **JSON** (.json)

### Microsoft Office Suite
- **Excel**: 2003+ (.xls, .xlt, .xlsb, .xlsm, .xlsx, .xltx)
- **PowerPoint**: 2003+ (.ppt, .pps, .pot, .pptm, .pptx, .ppsx, .sldx, .potx)
- **Word**: 2003+ (.doc, .rtf, .bak, .dot, .docm, .dotm, .dotx)
- **Visio** (.vsd, .vsdx, .vdx, .vsdm, .vss, .vssx, .vssm, .vtx, .vstm, .vdw, .vsr, .vstx, .vsx)
- **Outlook Messages** (.msg)

### Other Formats
- **COTI Packages** (.zip, .coti)
- **Image Files** (.bmp, .dib, .gif, .jpg, .jpeg, .jpe, .jif, .jfif, .jfi, .png, .tif, .tiff)
- **Java Properties** (.properties)
- **Include Files** (.inc)
- **Plain Text** (.txt, .inf, .ini, .reg)
- **PDF Files** (.pdf)
- **.NET Resource Files** (.xml, .resx)
- **OpenDocument** (.odf, .odt, .odp, .ods)
- **SubRip Subtitles** (.srt)
- **Multilingual Tables** (.csv, .tsv, .txt, .xlsx, .xls, .xlsm)
- **Multilingual XML** (.mulix)
- **ZIP Archives** (.zip)
- **Help Workshop** (.hhc, .hhk)
- **AuthorIT Projects** (.xml)
- **DITA Documents** (.dita, .xml, .ditamap, .bookmap)
- **FreeMind Mindmaps** (.mm)
- **SVG Drawings** (.svg)
- **Typo3 Pages** (.xml)
- **WordPress XLIFF/WPML** (.xlf, .xlif, .xliff)

## Bilingual Formats

- **PO Gettext Files** (.po)
- **SDL Trados Studio XLIFF** (.sdlxliff)
- **SDL Trados TagEditor** (.ttx)
- **Translation Memory Exchange** (.tmx)
- **Bilingual DOC/RTF Documents** (.doc, .rtf, .bak)
- **WordFast Pro/GlobalLink TransStudio TXML** (.txml)
- **XLIFF Files** (.xlf, .xlif, .xliff)
- **XLIFF:doc Files** (.xliffdoc)
- **Phrase (Memsource) XLIFF** (.mxliff)

## memoQ Bilingual Formats

- **memoQ XLIFF** (.mqxliff, .mqxlz)
- **Table RTF Documents** (.rtf)

## Package Formats

- **SDL Trados Studio Packages** (.sdlppx)
- **STAR Transit Packages** (.pxf, .ppf)
- **TIPP Packages**
- **memoQ Handoff Packages** (.mqout)
- **SDL World Server** (.xlz, .wsxz)

## Advanced Processing

You can work on or modify every file from the above list by using Regex tagger or a cascading filter, enabling custom processing beyond standard import/export capabilities.
